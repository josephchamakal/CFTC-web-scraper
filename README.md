FEATURES

Automated Data Extraction: Collects COT reports for 50+ futures contracts including commodities, currencies, and indices

Structured Data Processing: Parses raw text reports into clean, structured CSV formats

Error Handling: Robust parsing with comprehensive error handling and logging

DATA EXTRACTED

For each futures contract, the scraper extracts:

Contract specifications and open interest

Position data by trader category (Commercial, Non-Commercial, Nonreportable)

Weekly changes and percentage breakdowns

Number of traders in each category

TECH STACK

Python 3.x with Scrapy framework

Regex patterns for text processing

USE CASE

Quantitative trading research

Market sentiment analysis

Risk management

Academic research on futures markets

Trading strategy development
